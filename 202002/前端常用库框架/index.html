前端常用库、框架

发表于 2020-02-12

Vue.js

渐进式 JavaScript 框架 https://cn.vuejs.org/

1.x 由个人开发者尤雨溪开发 2.x..已经发展为团队

目前最火（关注多）

React

用于构建用户界面的 JavaScript 库 https://react.docschina.org/

一般我们还是叫做框架

2013 年由Facebook开源

目前最流行（使用多）

Angular

一套框架，多种平台 移动端 & 桌面端 https://angular.cn/

1.x 的版本叫 AngularJs 2.x 以后称 Angular

google 开源的

Bootstrap

  • Bootstrap 目前流行有两个版本

Bootstrap3

Bootstrap4

v3 和 v4 的区别

3

  • 使用 Less
  • pc 端多

4

  • 使用 Sass
  • flex 布局
  • 移动设备优先

Font Awesome

一套绝佳的图标字体库和 CSS 框架 http://fontawesome.dashgame.com/ 4.1.0 开始不再支持 IE7

Animate.css

一个 CSS3 动画库 https://daneden.github.io/animate.css/

MUI

https://dev.dcloud.net.cn/mui/ 最接近原生 APP 体验的高性能前端框架

View UI

一套基于 Vue.js 的高质量 UI 组件库 https://www.iviewui.com/

Vant

轻量、可靠的移动端 Vue 组件库 https://youzan.github.io/vant/#/zh-CN/

Element

Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库 https://element.eleme.cn/#/zh-CN

Layui

经典模块化前端框架 https://www.layui.com/

Jquery

https://www.Jquery.com/

UIkit

http://www.getuikit.net/ 一个轻量级的模块化前端框架， 用于快速开发强大的 Web 界面。

Weex

编写跨平台应用 https://weex.apache.org/zh/guide/ 阿里家的，于 2017-02-24 迁移至 Apache 基金会

Mint UI

基于 Vue.js 的移动端组件库 https://mint-ui.github.io/#!/zh-cn

uni-app

https://uniapp.dcloud.io/ > uni-app 是一个使用 Vue.js 开发所有前端应用的框架

React Native

使用 JavaScript 和 React 编写原生移动应用 https://reactnative.cn/

Flutter

谷歌的移动 UI 框架 https://flutterchina.club/ 需要学习 dart

Chart.js

http://chartjs.cn/ 基于 HTML5 的 JavaScript 图表库

ECharts

https://www.echartsjs.com/zh/index.html 使用 JavaScript 实现的开源可视化库

百度的

Highcharts

https://www.highcharts.com.cn/ 一个用纯 JavaScript 编写的一个图表库 在个人网站、学校网站及非盈利机构中使用 Highcharts 完全不需要经过我们的许可，直接可以任意使用！

商用要钱的

C3.js

https://c3js.org/ D3-based reusable chart library

D3.js

https://d3js.org/ 数据驱动的文档
